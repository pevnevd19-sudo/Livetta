# Livetta — восстановление всех обновлений + только правка размеров

Эта сборка сделана из последней полной рабочей версии проекта, а не из старого состояния GitHub.

## Восстановлено

- Админка с ролями.
- Склад товаров.
- Отдельный склад камней.
- Заказы.
- Excel-экспорт и импорт заказов.
- Статусы оплат.
- Ссылка на чек.
- Конструктор с выбором замка.
- Удаление пользователей с защитой последнего owner.
- Визуальные правки админки.
- Юридические страницы.
- Доставка, оплата, гарантия, возвраты.
- Карусель и популярные товары.

## Изменено только по запросу о размерах

- Размеры товаров заменены на XS / S / M / L / XL.
- Стандартная сетка:
  - XS — 38 см;
  - S — 40 см;
  - M — 42 см;
  - L — 44 см;
  - XL — 46 см.
- В админке у конкретного украшения можно изменить сантиметры для каждого размера.
- Для браслета можно поставить, например, XS = 16 см.
- Для категории «Серьги» размеры скрываются автоматически.
- Добавлена галочка «Не показывать размер».
- В карточке товара можно выбрать размер перед добавлением в корзину.
- Размер сохраняется в корзине, заказе и Excel.
- В конструкторе размер показывается как XS / S / M / L / XL с подписью в сантиметрах.
- Добавлена пометка: при заказе украшения с замком карабин есть удлинение 4 см.
- Исправлено ухудшение качества фото при наведении.

## Не трогалось

- База данных.
- .env.
- Загруженные фотографии.
- Архитектура проекта.
- Стрелки карусели.

## Установка на VPS

```bash
cd /var/www
cp -r livetta livetta_backup_before_restore
```

Распакуйте архив поверх `/var/www/livetta`.

Не заменяйте:

```text
.env
data/database.sqlite
uploads
data/livetta-orders.xlsx
```

После замены:

```bash
cd /var/www/livetta
npm install
pm2 restart livetta
```

В браузере нажмите `Ctrl + F5`.
