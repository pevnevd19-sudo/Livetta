# Что изменено

- Поля «Тип украшения», «Размер» и «Тип замка» выровнены в одну строку.
- Все три поля имеют одинаковую ширину и высоту.
- Подписи и выпадающие списки находятся на одном уровне.
- На телефонах поля аккуратно располагаются друг под другом.
- Из выбора типа украшения удалён «Браслет».
- В конструкторе теперь доступно только «Колье».
- Сервер также отклоняет старые заказы конструктора с типом «Браслет».
- Длина колье остаётся от 30 до 50 см.
- Архитектура и остальные функции не менялись.
- Стрелки карусели не менялись.

После замены:

```bash
npm install
npm start
```

Затем обновите страницу через `Ctrl + F5`.
