# Livetta — исправление визуала админки

## Что исправлено

- Карточки украшений, камней, заказов и пользователей теперь занимают полную ширину.
- Убрано старое правило, которое делило карточку камня на узкие колонки.
- Форма «Редактировать камень» полностью переразмечена:
  - добавлены подписи к каждому полю;
  - название и описание получили полную ширину;
  - цена, размер, остаток и цвет расположены аккуратной сеткой;
  - поле загрузки фотографии больше не выглядит как огромная кнопка;
  - чекбокс и кнопка сохранения получили нормальные отступы.
- Аналогично улучшена форма редактирования украшения.
- Исправлены размеры фотографий камней и украшений.
- Быстрое изменение склада адаптировано под большие и маленькие экраны.
- Формы пользователей перестали слипаться.
- Убраны старые минимальные размеры полей, из-за которых элементы выходили за карточки.
- Добавлена адаптация для планшетов и телефонов.

## Что не менялось

- Архитектура проекта.
- API и база данных.
- Расчёт конструктора.
- Онлайн-оплата и заказы.
- Складские операции.
- Роли пользователей.
- Параметры стрелок карусели:
  - назад: left 18px, padding-bottom 7px;
  - вперёд: right 18px, padding-bottom 7px.

## После замены

```bash
npm install
npm start
```

Затем обновите админку через `Ctrl + F5`.
